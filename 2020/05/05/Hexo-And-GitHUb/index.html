Hexo And GitHUb

Hexo是一个能通过解析Markdown生成博客页面的框架。

通过使用Github的github page不需要 域名即可可以线上访问。

过程

  • 安装node.js(建议使用nvm安装)。解压版点击install.cmd,修改settings.txt放置安装目录下
  • 安装Git下载,并设置github账号
  • github创建仓库
  • 生成ssh添加到github
  • 部署Hexo到GitHub，修改_config.yml最后一行
  • 使用deploy-git
